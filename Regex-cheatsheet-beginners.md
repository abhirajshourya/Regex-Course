# Anchors

|  Symbol  | Purpose |
| :--: | :-----------------------------------------------------: |
|  ^   | Start of string, or start of line in multi-line pattern |
|  $   |   End of string, or end of line in multi-line pattern   |

# Quantifiers

| Symbol |     Meaning      |
| :--:   | :--------------: |
|   *    |    0 or more     |
|   +    |    1 or more     |
|   ?    |      0 or 1      |
|  {5}   |    Exactly 5     |
|  {5,}  |    5 or more     |
| {5,10} | 5, 6,7,8,9 or 10 |

# Character Sets

| Symbol |      Purpose      |
| :----: | :---------------: |
|   \s   |    White Space    |
|   \S   |  Not White Space  |
|   \d   |       Digit       |
|   \D   |     Not Digit     |
|   \w   |       Word        |
|   \W   |     Not Word      |
|   \x   | Hexadecimal Digit |
|   \O   |    Octal Digit    |
